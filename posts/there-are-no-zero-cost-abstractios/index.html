There are no zero-cost abstractions

CppCon 2019: Chandler Carruth “There Are No Zero-cost Abstractions”

Key takeaways:

  • Tradeoff 所有的抽象都有代價，只有當抽象帶來的好處大於壞處時才值得
  • Run time v.s. build time （protocol buffer）
  • Trivia class (unique pointer v.s. raw pointer)
  • Human costs (a huge function v.s. multiple separated smaller functions, parameters)
